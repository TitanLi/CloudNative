# std.ssh

## 新增ssh_workflow
```yaml
$ vim ssh_workflow.yaml
version: '2.0'

ssh_workflow:
  input:
    - host
    - username
    - password

  tasks:
    task1:
      action: std.ssh host=<% $.host %> username=<% $.username %> \
              password=<% $.password %>
      input:
        cmd: "cd /home/ubuntu/ && ls"
      on-success: task2

    task2:
      action: std.echo output=<% task(task1).result %>
```

## 建立workflow
```shell
$ mistral workflow-create ssh_workflow.yaml
```

## 建立execution
