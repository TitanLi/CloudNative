# publish
## 新增workflow檔案
```yaml
$ vim publish-test.yaml

---
version: '2.0'

publish-test:
  tasks:
    publish-test:
      action: std.noop
      on-success:
        publish:
          global:
            my_var: "global value"
          branch:
            my_var: "branch value"
        next:
          - A
    A:
      action: std.echo output='global=><% global(my_var) %>  branch=><% $.my_var%>'

    B:
      # 等待前方task執行完畢，否則global my_var會是null
      wait-before: 2
      action: std.echo output='global=><% global(my_var) %>  branch=><% $.my_var%>'
```
